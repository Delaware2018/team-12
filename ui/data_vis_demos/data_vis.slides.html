Goodwill Rewards: Data Visualization Test for¶

Overview¶

In Goodwill's rewards-based donation web application, we use natural language processing to identity parameters regarding the (1) relation between user donations and (2) community employment levels - i.e. your donation of x amount will result in x benefit for this community.

Display Info¶

Linked responsive visualization plots composed of (a) radial bar graph from Bokeh and an (b) intertopic distance bubble plot from pyLDAvis - showing donation prevalance in different geographical, age, etc. groups. Additional display options could include: Basic Bubble Plot and/or Density Hexagon Map.
